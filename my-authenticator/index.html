My Authenticator

My Authenticator is a simple application to generate one-time passwords which can be used to log into web sites or other applications which support two-factor authentication.

Features

  • Add new accounts by scanning QR-codes
  • All data is securely stored in iOS KeyChain
  • Supports time-based one-time passwords
